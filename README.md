# 📝 MERN Blog System (Article App)
A full-stack blog management system built with the MERN stack (MongoDB, Express, React, Node.js). 

## ✨ Features
* **Full CRUD Operations:** Create, Read, Update, and Delete articles.
* **Dark Mode:** User-friendly UI with dark/light mode toggle.
* **Responsive Design:** Works perfectly on mobile, tablet, and desktop.
* **Fast & Lightweight:** Built using Vite for a modern development experience.

## 🚀 Tech Stack
* **Frontend:** React.js, Tailwind CSS (or CSS Modules).
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB with Mongoose.

## 🛠️ Installation & Setup
1. Clone the repository.
2. For the **Backend**:
   - Navigate to `/back` folder.
   - Run `npm install`.
   - Create a `.env` file based on `.env.example` and add your MongoDB URI.
   - Run `npm start`.
3. For the **Frontend**:
   - Navigate to `/front` folder.
   - Run `npm install`.
   - Run `npm run dev`.
